# Homepage Monorepo

A monorepo containing my personal homepage and blog.

## Structure

- `apps/homepage` - Main homepage
- `apps/blog` - Personal blog

## Development

Run all apps in development mode:

```bash
yarn dev
```

Run a specific app:

```bash
cd apps/homepage && yarn dev
cd apps/blog && yarn dev
```

## Build

Build all apps:

```bash
yarn build
```

## Tech Stack

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [GSAP](https://greensock.com/gsap/)
- [Turborepo](https://turbo.build/)
